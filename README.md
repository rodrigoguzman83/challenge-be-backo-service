* npm run dev

## API Reference
* GET: /users -> Returns a list of users
* GET: /users/:userId -> Returns a user by ID
* POST: /users/:userId -> Create a new user
* PUT: /users/:userID -> Update a user by ID
* DELETE: /users/:userId -> Delete a user by ID
